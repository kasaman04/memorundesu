🛠 ClaudeCode 指示書：ARメモアプリ「かざメモ（仮）」開発用
✅ 目的
カメラで物体をかざすと、その物体に紐づけたテキスト／写真／音声のメモを登録・表示できるアプリを構築する。
さらに、位置情報を組み合わせてメモを分類し、他ユーザーとの共有機能もサポートする。

📱 想定機能一覧（MVP構成）
1. 物体検出（Object Detection）
スマホのカメラを通してリアルタイムに物体を検出

同一物体を識別するため、物体の画像からハッシュまたは特徴量ベクトルを生成・保存

2. メモ登録機能
各物体に対して以下を紐づけて保存可能：

テキストメモ（textarea）

写真（画像添付）

音声（録音→再生）

緯度経度（自動取得）

3. メモの表示
再度カメラを同一物体にかざすと、既存メモをポップアップで表示

メモの編集・削除も可能

4. メモ完了機能
メモに「完了」ステータスを付与可能

完了済みメモは色分け表示

5. メモ共有機能（任意）
任意のメモをシェアリンク経由で他者に共有

または「家族グループ」などの共有スペースを作成し、その中で同期

🧱 使用技術例（想定）
項目	技術候補
フロント	Flutter / React Native（カメラ制御しやすいため）
物体認識	TensorFlow Lite / Firebase ML Kit
データ保存	SQLite / Firestore（オフライン + クラウド）
位置情報	Geolocation API（端末から取得）
音声録音・再生	WebAudio / Expo Audio（React Native）
画像添付	FileReader or Native Camera API
シェア機能	Firebase Auth + Firestore シェアリンク生成
